<h1 align="center">Welcome to Baemin-18 😄</h1>

## 프로젝트 개발내용

FE -

    > pug block, mixins 사용

    > 회원 가입 관련 정보 임시 저장은 sessionStorage 사용

    > 로딩 UI 적용
  
BE - 

    > nedb-promise 사용 (Embedded DB & 비동기 처리)

    > bcrypt 사용으로 안전한 password 저장

    > json web token으로 인증 토큰 발행

    > 토큰을 쿠키에 담아 메인페이지에서 유저의 권한을 식별
